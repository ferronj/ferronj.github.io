Nature of Code

As mentioned in my About page, several years ago, when looking for interesting programming projects to teach to an advanced group of middle school students. I came across Daniel Shiffman's Nature of Code and the Coding Train YouTube channel. I didn't wind up using much for that class, but I did use it myself to embrace the visual side of programming. A weakness of mine and something I found a lot of fun to experiment with.

Now that I have a website that I'm comfortable embedding a p5 JavaScript canvas into, I'd like to start getting some of those examples up here. So to start...

ok! Momentary celebration. I got that to work!

This is the starting point of the nature of code examples, "The seeker." From this point, we can manipulate forces and perception distances to create different behaviors. So in this example, the seeker follows the mouse. When giving a group of seekers the task of managing their positions and velocities with respect to the others in the group, we can start getting more interesting emergent properties.
